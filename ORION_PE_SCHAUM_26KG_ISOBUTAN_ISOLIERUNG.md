# PE-Schaum Thermoisolierung
## Spezifikation: 26 kg/m3 | Isobutan | Thermische Isolierung

---

## DIAGNOSE

Du arbeitest im **kritischsten Bereich**:

| Parameter | Dein Wert | Problem |
|-----------|-----------|---------|
| Dichte | 26 kg/m3 | Sehr niedrig = duenne Zellwaende |
| Treibmittel | Isobutan | Permeiert 12-15x schneller als Luft eindringt |
| Anwendung | Isolierung | Braucht geschlossene Zellen + Stabilitaet |

**Resultat:** Maximales Kollaps-Risiko nach Abkuehlung.

---

## LOESUNG 1: ADDITIV (Schnellste Umsetzung)

### Glycerolmonostearat (GMS)

```
Dosierung:     1.5-2.0 Gew.%
Einarbeitung:  Mit LDPE im Extruder bei 180C mischen
Wirkung:       Migriert zur Zelloberflaeche
               Bildet 1-4 um kristalline Barriere
               Verlangsamt Isobutan-Austritt
               Ermoeglicht Lufteintritt
```

**Bezugsquelle:**
- Lebensmittelchemie-Grosshandel (GMS E471)
- Croda, BASF, Evonik
- Ca. 3-5 EUR/kg

**Ergebnis:** Schrumpfung von 15-20% auf unter 2%

---

## LOESUNG 2: TREIBMITTEL-OPTIMIERUNG

### Option A: Isobutan + n-Butan Mischung (50/50)

```
Vorteil:       Permeiert langsamer als reines Isobutan
               Alterungszeit 5 Tage statt 12 Tage
               Schrumpfung 1.25% statt 1.56%
Nachteil:      Zwei Gase dosieren
```

### Option B: CO2 + N2 statt Isobutan (Gruen)

```
Vorteil:       KEINE Schrumpfung
               Keine Explosionsgefahr
               Keine Alterungszeit
               Umweltfreundlich
Nachteil:      Prozessumstellung noetig
               Andere Expansionsparameter
```

---

## LOESUNG 3: POLYMER-BLEND

### LDPE + LLDPE (90/10)

```
Mischung:      90% LDPE + 10% LLDPE
Effekt:        Erleichtert Zellnukleierung
               Schnellerer Gas-Austausch
               Stabiler bei CO2 als Treibmittel
```

---

## ORION EMPFEHLUNG FUER DICH

### Sofort-Loesung (ohne Prozessaenderung):

```
REZEPTUR FUER 26 kg/m3 ISOLIERSCHAUM
=====================================

Polymer:       LDPE (MFI 2-4 g/10min)
               
Additive:
- GMS:         1.8 Gew.%
- Talkum:      1.5 Gew.%  (Nukleierung)

Treibmittel:   Isobutan 6-7 pph

Prozess:
1. LDPE + GMS + Talkum bei 180C mischen
2. Isobutan einspritzen
3. Schaeumen bei 108-115C
4. SOFORT Wasserkuehlung (unter 10 Sekunden)
5. 48h lagern bei 20-25C vor Verwendung

Erwartetes Ergebnis:
- Dichte: 25-28 kg/m3
- Schrumpfung: <2%
- Zellstruktur: geschlossen, stabil
```

---

### Langzeit-Loesung (beste Qualitaet):

```
PREMIUM REZEPTUR
================

Polymer:       LDPE 90% + LLDPE 10%

Additive:
- GMS:         1.5 Gew.%
- Talkum:      2.0 Gew.%

Treibmittel:   CO2 + N2 (70/30) statt Isobutan

Vorteile:
- Keine Schrumpfung
- Keine Explosionsgefahr
- Keine Alterungszeit
- Sofort verwendbar
- Bessere Langzeit-Stabilitaet
```

## WAERMELEITUNG (Fuer Isolierung wichtig)

Bei 26 kg/m3 erreichst du:

| Parameter | Wert |
|-----------|------|
| Waermeleitfaehigkeit | 0.033-0.038 W/(m*K) |
| Zellgroesse optimal | 200-400 um |
| Geschlossene Zellen | >90% |

**Zum Vergleich:**
- Styropor EPS: 0.035-0.040 W/(m*K)
- Mineralwolle: 0.035-0.045 W/(m*K)
- PU-Hartschaum: 0.022-0.028 W/(m*K)

Dein PE-Schaum ist gut fuer Isolierung, aber nicht Spitze.
Fuer beste Isolierung waere PU besser (aber teurer + komplexer).

---

## WENN ES IMMER NOCH KOLLABIERT

Eskalationsstufen:

1. GMS auf 2.5% erhoehen
2. Talkum auf 3% erhoehen
3. Isobutan auf 5.5 pph reduzieren (hoehere Dichte akzeptieren)
4. Auf CO2/N2 umstellen
5. Leichte Vernetzung (0.5% DCP)

---

## ZUSAMMENFASSUNG

**Dein Problem:** 26 kg/m3 + Isobutan = maximales Kollaps-Risiko

**Schnellste Loesung:**
```
+1.8% GMS + 1.5% Talkum + schnelle Kuehlung + 48h Alterung
```

**Beste Loesung:**
```
LDPE/LLDPE 90/10 + CO2/N2 statt Isobutan = kein Kollaps
```
